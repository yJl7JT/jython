### jython install
- [dowload jython](https://www.jython.org/download)
- install jython
```
java -jar jython_installer-2.7.1.jar
```

### Add dependency
- maven dependency
```
<dependency>
    <groupId>org.python</groupId>
    <artifactId>jython-standalone</artifactId>
    <version>2.7.0</version>
</dependency>
```
- add jar (jython-2.7.1/jython.jar)
